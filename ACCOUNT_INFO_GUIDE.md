# Account Information Guide

## ภาพรวม

ระบบข้อมูลบัญชี (Account Info) ช่วยให้ผู้ใช้สามารถดูข้อมูลบัญชี โควต้า และการตั้งค่าต่างๆ ของ Slip2Go API ได้อย่างครบถ้วน

## ฟีเจอร์หลัก

### 1. การดึงข้อมูลบัญชี
- **Endpoint**: `GET https://connect.slip2go.com/api/account/info`
- **Authentication**: Bearer Token
- **การใช้งาน**: ดึงข้อมูลบัญชีและโควต้าปัจจุบัน

### 2. ข้อมูลที่แสดง
- **ข้อมูลร้าน**: ชื่อร้าน, แพ็กเกจ, วันหมดอายุ
- **โควต้าการตรวจสอบสลิป**: จำนวนครั้งที่เหลือ
- **โควต้าการสร้าง QR Code**: จำนวนครั้งที่เหลือ
- **เครดิตคงเหลือ**: จำนวนเครดิตสำหรับการตรวจสอบสลิปส่วนเกิน
- **การตั้งค่า**: ต่ออายุอัตโนมัติ, ตรวจสลิปด้วยเครดิต

### 3. การจัดการข้อมูล
- **รีเฟรชข้อมูล**: อัปเดตข้อมูลล่าสุด
- **แสดงสถานะ**: สถานะโควต้าและแพ็กเกจ
- **แจ้งเตือน**: แจ้งเตือนเมื่อโควต้าใกล้หมดหรือแพ็กเกจหมดอายุ

## การใช้งาน

### 1. เข้าสู่หน้าข้อมูลบัญชี
```
/account-info
```

### 2. ดูข้อมูลบัญชี
- ข้อมูลจะถูกโหลดอัตโนมัติเมื่อเข้าหน้า
- คลิก "รีเฟรช" เพื่ออัปเดตข้อมูลล่าสุด

### 3. ตรวจสอบสถานะ
- **โควต้าการตรวจสอบสลิป**: ดูจำนวนครั้งที่เหลือ
- **โควต้าการสร้าง QR Code**: ดูจำนวนครั้งที่เหลือ
- **เครดิตคงเหลือ**: ดูจำนวนเครดิตที่เหลือ

## ข้อมูลที่แสดง

### ข้อมูลร้าน
- **ชื่อร้าน**: ชื่อร้านค้าที่ลงทะเบียน
- **แพ็กเกจ**: แพ็กเกจปัจจุบัน (เช่น BASIC-1)
- **วันหมดอายุ**: วันที่แพ็กเกจจะหมดอายุ
- **สถานะ**: แสดงสถานะแพ็กเกจ (หมดอายุ/ใกล้หมดอายุ/ปกติ)

### โควต้าการตรวจสอบสลิป
- **จำนวนทั้งหมด**: จำนวนครั้งที่สามารถตรวจสอบสลิปได้
- **ใช้แล้ว**: จำนวนครั้งที่ใช้ไปแล้ว
- **คงเหลือ**: จำนวนครั้งที่เหลือ
- **เปอร์เซ็นต์**: เปอร์เซ็นต์การใช้งาน
- **สถานะ**: สีแสดงสถานะ (เขียว/เหลือง/แดง)

### โควต้าการสร้าง QR Code
- **จำนวนทั้งหมด**: จำนวนครั้งที่สามารถสร้าง QR Code ได้
- **ใช้แล้ว**: จำนวนครั้งที่ใช้ไปแล้ว
- **คงเหลือ**: จำนวนครั้งที่เหลือ
- **เปอร์เซ็นต์**: เปอร์เซ็นต์การใช้งาน
- **สถานะ**: สีแสดงสถานะ (เขียว/เหลือง/แดง)

### เครดิตคงเหลือ
- **จำนวนเครดิต**: จำนวนเครดิตที่เหลือ
- **การใช้งาน**: ใช้สำหรับการตรวจสอบสลิปส่วนเกิน
- **สถานะ**: แสดงสถานะเครดิต

### การตั้งค่าแพ็กเกจ
- **ต่ออายุอัตโนมัติ**: การตั้งค่าต่ออายุแพ็กเกจอัตโนมัติ
- **ตรวจสลิปด้วยเครดิต**: การตั้งค่าตรวจสลิปส่วนเกินด้วยเครดิต

## การแสดงผล

### Progress Bar
- แสดงเปอร์เซ็นต์การใช้งานโควต้า
- สีเปลี่ยนตามสถานะ:
  - **เขียว**: มากกว่า 50%
  - **เหลือง**: 20-50%
  - **แดง**: น้อยกว่า 20%

### Status Indicators
- **CheckCircle**: เปิดใช้งาน
- **XCircle**: ปิดใช้งาน
- **AlertTriangle**: คำเตือน

### Color Coding
- **เขียว**: สถานะดี
- **เหลือง**: คำเตือน
- **แดง**: วิกฤต

## การจัดการข้อผิดพลาด

### Error Handling
- **ไม่พบข้อมูล**: แสดงข้อความ "ไม่สามารถดึงข้อมูลบัญชีได้"
- **ข้อผิดพลาด API**: แสดงข้อความข้อผิดพลาดจาก API
- **ข้อผิดพลาดเครือข่าย**: แสดงข้อความ "เกิดข้อผิดพลาดในการดึงข้อมูลบัญชี"

### User Feedback
- แสดง Loading state ขณะดึงข้อมูล
- แสดง Toast notification สำหรับการดำเนินการ
- แสดง Alert สำหรับข้อผิดพลาดและคำเตือน

## การใช้งานในระบบ

### 1. ตรวจสอบสถานะบัญชี
- ไปที่หน้า "ข้อมูลบัญชี" (/account-info)
- ดูข้อมูลบัญชีและโควต้าปัจจุบัน
- ตรวจสอบสถานะแพ็กเกจและโควต้า

### 2. การจัดการโควต้า
- ตรวจสอบโควต้าการตรวจสอบสลิป
- ตรวจสอบโควต้าการสร้าง QR Code
- ดูเครดิตคงเหลือ

### 3. การตั้งค่า
- ตรวจสอบการตั้งค่าต่ออายุอัตโนมัติ
- ตรวจสอบการตั้งค่าตรวจสลิปด้วยเครดิต

## ข้อดีของระบบ

### 1. ความชัดเจน
- แสดงข้อมูลครบถ้วนและชัดเจน
- ใช้สีและไอคอนช่วยในการแสดงผล
- แสดงสถานะแบบ Real-time

### 2. ความสะดวก
- ข้อมูลอัปเดตอัตโนมัติ
- รีเฟรชข้อมูลได้ง่าย
- แสดงข้อมูลที่สำคัญในที่เดียว

### 3. การแจ้งเตือน
- แจ้งเตือนเมื่อโควต้าใกล้หมด
- แจ้งเตือนเมื่อแพ็กเกจหมดอายุ
- แสดงสถานะที่เข้าใจง่าย

## การพัฒนาเพิ่มเติม

- ต่ออายุแพ็กเกจ

- ดูประวัติการต่ออายุ

- รายงานการใช้งานเครดิต

### 3. การแจ้งเตือน
- แจ้งเตือนเมื่อโควต้าใกล้หมด
- แจ้งเตือนเมื่อแพ็กเกจใกล้หมดอายุ
- แจ้งเตือนการต่ออายุ

## การเชื่อมต่อกับระบบอื่น

### 1. Slip Verification
- ใช้โควต้าการตรวจสอบสลิป
- ใช้เครดิตสำหรับการตรวจสอบส่วนเกิน

### 2. QR Code Generation
- ใช้โควต้าการสร้าง QR Code
- ตรวจสอบโควต้าก่อนสร้าง

### 3. Payment System
- เชื่อมต่อกับระบบชำระเงิน
- ใช้สำหรับการต่ออายุแพ็กเกจ

## สรุป

ระบบข้อมูลบัญชีเป็นเครื่องมือสำคัญสำหรับการจัดการบัญชี Slip2Go โดยให้ข้อมูลที่ครบถ้วน ชัดเจน และใช้งานง่าย ทำให้ผู้ใช้สามารถจัดการบัญชีและโควต้าได้อย่างมีประสิทธิภาพ

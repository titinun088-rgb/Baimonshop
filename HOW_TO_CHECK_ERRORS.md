# วิธีตรวจสอบ Error

## เปิด Developer Console

### Google Chrome / Microsoft Edge:
1. กด `F12` หรือ `Ctrl + Shift + I`
2. เลือกแท็บ **Console**
3. ดูข้อความ Error (ข้อความสีแดง)
4. Copy ข้อความทั้งหมดมาให้

### Firefox:
1. กด `F12` หรือ `Ctrl + Shift + K`
2. ดูข้อความ Error
3. Copy มาให้

## สิ่งที่ควรส่งมา:
1. ข้อความ Error ทั้งหมด (สีแดง)
2. Screenshot ถ้าทำได้
3. บอกว่ากำลังทำอะไร (Import Sales หรือ Game Items)
4. ไฟล์ Excel ที่ใช้ (ถ้าทำได้)

## ตัวอย่าง Error ที่พบบ่อย:

### Error 1: "Cannot read properties..."
→ ปัญหาโค้ด (ผมแก้ให้)

### Error 2: "ไม่พบเกม..."
→ ชื่อไม่ตรง (ต้อง copy จากระบบ)

### Error 3: "ข้อมูลไม่ครบถ้วน"
→ มีช่องที่จำเป็นเว้นว่าง

### Error 4: "ราคาต้องมากกว่า 0"
→ ใส่ข้อความแทนตัวเลข

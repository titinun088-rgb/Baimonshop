# 🎮 คู่มือการใช้งาน: เพิ่มรายการขายในเกม

## ✅ ระบบพร้อมใช้งานแล้ว!

เมื่อคลิกเข้าไปในตัวเกม คุณสามารถเพิ่มรายการที่ขายได้ ระบบจะคำนวณกำไรให้อัตโนมัติ!

---

## 🚀 วิธีใช้งาน (Step by Step)

### **ขั้นตอนที่ 1: เข้าหน้ารายละเอียดเกม**

1. เปิดเว็บแล้วไปที่หน้า **"เกม"** (Games)
2. คลิกที่ **การ์ดเกมที่ต้องการ**
3. ✅ เข้าสู่หน้ารายละเอียดเกม (จะแสดง URL: `/games/[id]`)

---

### **ขั้นตอนที่ 2: เพิ่มรายการขาย**

1. ในหน้ารายละเอียดเกม คลิกปุ่ม **"เพิ่มรายการเติม"** (สีม่วง มุมขวาบน)
2. ระบบจะเปิด Dialog ให้กรอกข้อมูล:

   **📝 ฟอร์มที่ต้องกรอก:**
   
   - **ชื่อรายการ*** (บังคับ)
     - ตัวอย่าง: `VP 1000`, `Diamond 500`, `UC 1200`
   
   - **ราคาทุน (฿)*** (บังคับ)
     - ตัวอย่าง: `200`
     - ราคาที่เราซื้อมา
   
   - **ราคาขาย (฿)*** (บังคับ)
     - ตัวอย่าง: `250`
     - ราคาที่เราขาย
   
   - **URL รูปภาพ** (ไม่บังคับ)
     - ตัวอย่าง: `https://example.com/vp1000.jpg`
     - ใส่หรือไม่ใส่ก็ได้

3. ✅ **ระบบคำนวณกำไรอัตโนมัติ!**
   - จะแสดงกล่องสีเขียว: `กำไร: ฿50.00 (25%)`
   - ถ้าขาดทุน จะแสดงสีแดง และเตือน!

4. คลิกปุ่ม **"เพิ่มรายการ"**

5. ✅ **เสร็จแล้ว!** รายการจะปรากฏในตารางทันที

---

## 🖼️ การแสดงผลในตาราง

เมื่อเพิ่มรายการแล้ว จะปรากฏในตารางแบบนี้:

```
┌────────────────────────────────────────────────────────────────────┐
│  ชื่อรายการ    │ ราคาทุน │ ราคาขาย │  กำไร   │ % กำไร │ จัดการ │
├────────────────────────────────────────────────────────────────────┤
│ 🖼️ VP 1000     │  ฿200.00 │  ฿250.00 │ ฿50.00  │  25%   │   ⋮   │
│ 🖼️ UC 1200     │  ฿150.00 │  ฿180.00 │ ฿30.00  │  20%   │   ⋮   │
│    Diamond 500 │  ฿100.00 │  ฿130.00 │ ฿30.00  │  30%   │   ⋮   │
└────────────────────────────────────────────────────────────────────┘
```

### **📸 รูปภาพ:**
- ✅ ถ้าใส่ URL รูป → จะแสดง **รูปเล็กๆ ขนาด 32x32px** หน้าชื่อรายการ
- ❌ ถ้าไม่ใส่ URL → จะแสดงแค่ชื่อ (ไม่มีรูป)

### **💰 กำไร:**
- 🟢 สีเขียว = กำไร (ค่าบวก)
- 🔴 สีแดง = ขาดทุน (ค่าลบ)

## ✨ Features พิเศษ

### **1. คำนวณกำไรอัตโนมัติ**

พิมพ์ราคาทุน + ราคาขาย → ระบบคำนวณให้ทันที!

```
ราคาทุน: 200
ราคาขาย: 250

→ กำไร: ฿50.00 (25%)
```

### **2. แสดงสถิติ**

ด้านบนจะมีสถิติรวม:
- 📊 จำนวนรายการทั้งหมด
- 💰 กำไรเฉลี่ย
- 📈 กำไรสูงสุด
- 📉 กำไรต่ำสุด

### **3. เตือนเมื่อขาดทุน**

ถ้าราคาขาย < ราคาทุน:
- ⚠️ แสดงแถบสีแดง
- ⚠️ แจ้งเตือน "ราคาขายต่ำกว่าราคาทุน คุณจะขาดทุน!"
- ✅ แต่ยังสามารถบันทึกได้ (กรณีลดราคาขาย)

### **4. รูปภาพ Optional**

- ✅ **มีรูป:** แสดงรูปเล็กๆ หน้าชื่อรายการ
- ✅ **ไม่มีรูป:** แสดงแค่ชื่อ (ยังใช้งานได้ปกติ)

## 🔧 การจัดการรายการ

### **แก้ไขรายการ:**

1. คลิก **⋮** (เมนู) ที่ปลายแถว
2. เลือก **"แก้ไข"**
3. แก้ไขข้อมูล (ชื่อ, ราคา, รูป)
4. คลิก **"บันทึกการเปลี่ยนแปลง"**

### **ลบรายการ:**

1. คลิก **⋮** (เมนู) ที่ปลายแถว
2. เลือก **"ลบ"** (สีแดง)
3. ยืนยันการลบ
4. ✅ รายการถูกลบถาวร

## 💡 Tips & Tricks

### **1. การตั้งราคาที่ดี**

```
ราคาทุน: 200
ราคาขาย: 250
กำไร: 25%

✅ กำไร 20-30% = เหมาะสม
✅ กำไร 30-50% = ดี
⚠️ กำไร < 10% = น้อยไป
```

### **2. ใส่รูปหรือไม่ใส่?**

**ใส่รูป (แนะนำ):**
- ✅ ลูกค้าเห็นภาพชัดเจน
- ✅ ดูเป็นมืออาชีพ
- ✅ ขายได้ง่ายขึ้น

**ไม่ใส่รูป:**
- ✅ ประหยัดเวลา
- ✅ ใช้งานได้ปกติ

### **3. ตั้งชื่อรายการให้ชัดเจน**

✅ **ดี:**
- `VP 1000 + โบนัส 100`
- `Diamond 500 (ของแท้)`
- `UC 1200 เติมเร็ว 1 นาที`

❌ **ไม่ดี:**
- `รายการ 1`
- `test`
- `abc`

### **4. URL รูปภาพ**

**แหล่งรูปฟรี:**
- Unsplash: `https://source.unsplash.com/200x200/?valorant`
- Imgur: อัปโหลดแล้วคัดลอก URL
- Direct link: คลิกขวา → Copy image address

## 🐛 Troubleshooting

### **ปัญหา: กดเพิ่มรายการแล้วไม่เห็น Dialog**

**แก้ไข:**
1. ตรวจสอบว่าอยู่ในหน้ารายละเอียดเกม
2. Refresh หน้าเว็บ
3. ตรวจสอบว่ามีสิทธิ์แก้ไข (เจ้าของเกมหรือ Admin)

### **ปัญหา: รูปไม่แสดง**

**แก้ไข:**
1. ตรวจสอบ URL รูปภาพถูกต้อง
2. URL ต้องเป็น direct link (ลงท้าย .jpg, .png)
3. เปิด URL ใน browser ทดสอบ

### **ปัญหา: ไม่มีปุ่มเพิ่มรายการ**

**สาเหตุ:**
- เกมไม่ใช่ของคุณ (ถ้าเป็น Seller)
- ไม่ได้ login

**แก้ไข:**
- Login ใหม่
- ใช้ Admin account
- เลือกเกมของตัวเอง
